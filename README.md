## Install

[npm][]:

```sh
npm install @ladjs/email
```

[yarn][]:

```sh
yarn add @ladjs/email
```


## Usage

```js

const @ladjs/email = new @ladjs/email();

console.log(@ladjs/email.renderName());
// script
```
